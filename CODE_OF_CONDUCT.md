# Código de Conduta

Adotamos o [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Escopo Específico

Este projeto é **ferramenta de pesquisa em segurança de LLMs**. Contribuições devem:

- Ser usadas apenas em contexto de pesquisa e desenvolvimento de defesas
- Não ser aplicadas para manipulação de modelos de produção sem autorização
- Respeitar termos de uso dos modelos de linguagem utilizados

Pesquisa responsável é bem-vinda. Uso malicioso não.

## Padrões Esperados

- Linguagem acolhedora
- Respeito por pontos de vista diferentes
- Crítica construtiva
- Empatia

## Inaceitável

- Trolling, insultos, ataques
- Uso do projeto para assediar indivíduos
- Publicação de dados privados

Reporte abusos ao mantenedor.
